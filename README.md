# 🦠 COVID-19 & Vaccination Analysis Dashboard

An interactive data visualization dashboard built with **Python** and **Streamlit** to analyze COVID-19 cases and vaccination progress in India. The dashboard provides insightful charts, data cleaning features, and interactive visualizations to help users explore pandemic trends effectively.

---

## 📌 Project Overview

The COVID-19 & Vaccination Analysis Dashboard is designed to transform raw COVID-19 datasets into meaningful visual insights. It enables users to clean data, analyze pandemic statistics, monitor vaccination progress, and interact with dynamic charts through an easy-to-use web interface.

---

## ✨ Features

* 🏠 Professional Home Page
* 🧹 Data Cleaning and Preprocessing
* 📊 COVID-19 Case Analysis
* 💉 Vaccination Analysis

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/covid-dashboard.git
```

Move into the project folder:

```bash
cd covid-dashboard/cov
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run Dashboard.py
```
